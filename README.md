# Hungry-Spoon

Hungry-Spoon is a recipe-finder web application that allows users to search for recipes, view detailed ingredients and instructions, and explore meal ideas with a clean, responsive interface.

**Live Demo:** https://hungry-spoon.netlify.app/

**Features:**
- Search recipes by name or ingredients
- View recipe details with ingredients and instructions
- Responsive, user-friendly design
- Easy to extend and customize

**How to Use:**
1. Clone the repository: `git clone https://github.com/Muhammad-Ali-16/hungry-spoon.git`
2. Install dependencies : `npm install`
3. Run the app: `npm run dev`
4. Search and explore recipes

**Project Files:**
- `index.html` — main UI
- `style.css` — styling
- `app.js` — handles search and recipe display
- Assets — images/icons used in the app
